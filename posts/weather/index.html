Weather

 12.09.2022 

Weather

날씨 = weather
온도 = temperature

해 = sun
구름 = cloud
비 = rain
눈 = snow
바람 = wind
번개 = lightening
천둥 = thunder

해나요 = sunny
따듯해요 = warm
더워요 = hot
흐려요 = cloudy
바람 불어요 = rainy
추워요 = cold

Seasons

봄 = spring
여름 = summer
가을 = fall
겨울 = winter

Small Talk

비가 와요 = it’s raining
눈이 와요 = it’s snowing
하늘이 맑습니다 = the sky is clear
날씨가 나빠요 = the weather is bad
날씨가 춥지요? = the weather is cold, right?
날씨가 영 안 좋아요 = the weather is really not good.
날씨가 더웠지요? = the weather is hot, right?
바람이 많이 불어요 = It’s windy
